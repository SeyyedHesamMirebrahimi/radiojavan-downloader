# RadioJavan Downloader
A simple google chrome extension to add download buttons to [RadioJavan](https://www.radiojavan.com/)'s tracks and podcasts.

<br />

## How to use?
1. Download the project by [this link](https://github.com/soroushchehresa/radiojavan-downloader/archive/master.zip) and then extract it.
2. Go to `chrome://extensions/` and turn on developer mode.
3. Click to the `Load unpacked` button then select the extracted folder to install the extension.
